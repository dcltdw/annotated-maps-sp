# A1.3 Polish (e2e + bundle split + i18n + XSS hardening) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Harden the merged A1.3 map slice along four axes the holistic review flagged — a real-browser end-to-end test of the "Viewing as" differentiator (the maplibre-render blind spot), maplibre bundle code-splitting, routing the UI strings through i18n, and HTML-escaping user-controlled fields in the map popups.

**Architecture:** Four mostly-independent improvements, ordered so each builds on the last: (1) escape popup HTML, (2) move UI chrome to i18n, (3) lazy-load `MapView` so maplibre splits into its own chunk, (4) a Playwright e2e that drives the real map with the API and map-style network-stubbed (deterministic, no backend/DB/tiles needed in CI). The frontend stays the visibility *consumer*; nothing changes the backend or the visibility engine.

**Tech Stack:** React + TypeScript (Vite PWA), maplibre-gl, react-i18next, Vitest + Testing Library, **@playwright/test** (new), GitHub Actions.

**Scope decisions (flag for sign-off at plan review):**
- **i18n direction = wire-through**, not removal — the design spec's intent was "i18n structure ready, English only," so this fulfils that promise rather than ripping it out. (If you'd rather remove i18n entirely, say so and Task 2 inverts.)
- **e2e boundary = real frontend + stubbed API/style.** The Playwright test exercises the *real maplibre render + marker/popup/switcher interaction in a browser* (the actual blind spot) with the API responses and the OpenFreeMap style intercepted, so it's deterministic and needs no backend, DB, or external tiles in CI. Full-stack e2e (live Django + seed) is a heavier future option; the API contract is already covered by the backend suite + the manual end-to-end done during A1.3.

- [ ] **Step 3: Create `src/lib/escapeHtml.ts`:**
```ts
const ENTITIES: Record<string, string> = {
  "&": "&amp;",
  "<": "&lt;",
  ">": "&gt;",
  '"': "&quot;",
  "'": "&#39;",
};

// Single-pass replace: each metacharacter maps directly to its entity, so the
// "&" rule never re-encodes the "&" inside an entity it just emitted.
export function escapeHtml(value: string): string {
  return value.replace(/[&<>"']/g, (ch) => ENTITIES[ch]);
}
```

- [ ] **Step 4: Run it; confirm it PASSES.**

- [ ] **Step 5: Use it in `peekHtml` and export `peekHtml`** (`src/components/MapView.tsx`). Add the import and rewrite `peekHtml`; update the FIXME:

```ts
// User-controlled fields (title, content, group/rule labels) are HTML-escaped
// before being injected into the popup. colorFor() returns only fixed hex codes.
export function peekHtml(note: NoteOut): string {
  const rows = note.sections
    .map((s) => {
      const label = escapeHtml(s.rule_label);
      return s.visibility === "teaser"
        ? `<span style="color:${colorFor(s.rule_type)}">🔒 ${label}</span>`
        : `<span style="color:${colorFor(s.rule_type)}">● ${label}</span> ${escapeHtml(s.content ?? "")}`;
    })
    .join("<br>");
  return `<div style="font:13px system-ui;line-height:1.45;color:#111"><b>${escapeHtml(note.title)}</b><br>${rows}</div>`;
}
```

```ts
import { expect, test } from "@playwright/test";

// A minimal valid maplibre style so the map loads without hitting OpenFreeMap.
const MINIMAL_STYLE = {
  version: 8,
  sources: {},
  layers: [{ id: "bg", type: "background", paint: { "background-color": "#e8e8e8" } }],
};

const MAP = { id: "m1", name: "Greater Boston", lng: -71.0589, lat: 42.3601, zoom: 12 };
const VIEWERS = [{ id: "owner", display_name: "You (owner)", reputation: 100 }];

const GUEST = [
  { id: "s1", order: 0, visibility: "visible", content: "scenic loop", rule_type: "public", rule_label: "Public" },
  { id: "s2", order: 1, visibility: "teaser", content: null, rule_type: "audience", rule_label: "Running club" },
];
const OWNER = [
  { id: "s1", order: 0, visibility: "visible", content: "scenic loop", rule_type: "public", rule_label: "Public" },
  { id: "s2", order: 1, visibility: "visible", content: "club fountain", rule_type: "audience", rule_label: "Running club" },
  { id: "s3", order: 2, visibility: "visible", content: "trusted tip", rule_type: "attribute_gate", rule_label: "Reputation ≥ 50" },
  { id: "s4", order: 3, visibility: "visible", content: "knee reminder", rule_type: "private", rule_label: "Private" },
];
const note = (sections: unknown[]) => [{ id: "n1", title: "Castle Island", lng: -71.0136, lat: 42.338, sections }];

test("the Viewing-as switcher re-filters a note's sections live", async ({ page }) => {
  await page.route("**/styles/positron**", (r) => r.fulfill({ json: MINIMAL_STYLE }));
  await page.route("**/api/v1/maps", (r) => r.fulfill({ json: [MAP] }));
  await page.route("**/api/v1/maps/*/viewers", (r) => r.fulfill({ json: VIEWERS }));
  await page.route("**/api/v1/maps/*/notes**", (route) => {
    const previewAs = new URL(route.request().url()).searchParams.get("preview_as");
    route.fulfill({ json: note(previewAs ? OWNER : GUEST) });
  });

  await page.goto("/");

  // real maplibre places a marker once the (stubbed) style loads
  const marker = page.locator(".maplibregl-marker").first();
  await expect(marker).toBeVisible();
  await marker.click();

  // Guest: Public visible, Running-club locked, gate/private absent
  await expect(page.getByText("scenic loop")).toBeVisible();
  await expect(page.getByText(/Locked/i)).toBeVisible();
  await expect(page.getByText(/Reputation ≥ 50/)).toHaveCount(0);

  // Switch persona → backend (stub) returns the owner slice → panel re-renders live
  await page.getByRole("button", { name: "You (owner)" }).click();
  await expect(page.getByText(/Reputation ≥ 50/)).toBeVisible();
  await expect(page.getByText("Private")).toBeVisible();
  await expect(page.getByText("club fountain")).toBeVisible();
});
```

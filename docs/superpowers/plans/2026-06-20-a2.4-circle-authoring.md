# A2.4 — Region Notes: Circle Authoring — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Let a user draw a **circle** (center + radius) and save it as a region note. A circle is **only a drawing tool** — the `ShapeDrawer` adapter emits a many-sided **polygon**, so a circle note is stored/rendered as an `area` (polygon) exactly like a freehand area. This is the cheapest A2 layer: a third draw-mode entry.

**Architecture:** Purely a MapScreen UI addition: a "Draw circle" button arms `drawMode="circle"`. terra-draw's circle mode (wired in A2.2b's adapter, **confirmed to emit a Polygon**) produces a `DrawShape{kind:"polygon"}`, which flows through the existing editor→`createNote`→`area`→render path unchanged. No backend/editor/adapter change.

**Tech Stack:** React + TS; the `ShapeDrawer` port (`TerraDrawCircleMode` already wired + circle→polygon confirmed in A2.2b's spike).

**Spec:** `docs/superpowers/specs/2026-06-19-a2-region-notes-design.md` (§6 circle layer; "circle is only a drawing tool that emits a polygon"). PR **A2.4** — **stacked on A2.3** (branch off `a2.3-line-authoring`; merge A2.3 first). Targets `main` once A2.3 lands.

**Conventions:** frontend from `frontend/`. Gates (REAL exit codes, separately — never `cmd | tail; echo $?`): `npm run test -- --run; echo $?` / `npm run lint; echo $?` / `npx tsc -b; echo $?` / `npm run build; echo $?` / `npm run e2e; echo $?`.

MapScreen draw entry (after A2.3): `handleStartDraw(mode: DrawMode)` arms `drawMode`; "Draw area" (`"polygon"`) + "Draw route" (`"line"`) buttons; the region-edit redraw uses `editorShape?.kind === "line" ? "line" : "polygon"` (a circle note is stored as a polygon, so it redraws as polygon — correct/intended). `DrawShape.kind` is only `"polygon"|"line"` (no circle kind — circle emits polygon).

- [ ] Frontend gates green (test/lint/tsc/build) + an e2e drives the draw-circle→save flow via the injected fake.

## Out of scope

Storing "this polygon was a circle" / re-editing as a circle (a circle is a one-way drawing tool → polygon, by design); vertex editing; true geometric circles (we use N-gon polygons).
